Refonte de la page d'accueil pour refléter la généralisation de l'application à tous les sports. L'application s'appelle désormais EloSports Manager et supporte la gestion de clubs pour différentes disciplines (frisbee, football, basketball, etc.) avec classement Elo. La page d'accueil présente le projet comme open source et gratuit, explique la sécurité des données via Supabase PostgreSQL avec Row Level Security (RLS), et inclut un lien vers le projet GitHub.
